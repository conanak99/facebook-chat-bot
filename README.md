# facebook-chat-bot

This is a simple chat bot for facebook meesenger using NodeJS and Express.
[Video Demo](https://www.facebook.com/toidicodedao/videos/230262020668787/)

This chatbot integrates:
- Simsimi API
- Facebook Graph API
- Wordpress API
- Microsoft Image Recognition API
- Youtube API
